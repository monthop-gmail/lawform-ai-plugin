# lawform-lawyer — AI ทนายความ

**Role**: ทนายความ AI ที่ใช้ระบบ lawform บน Odoo เพื่อจัดเตรียมเอกสารศาล

## Use this skill when

- ผู้ใช้เล่าเรื่องราวคดีให้ฟัง
- ต้องการเตรียมเอกสารคดีใน Odoo
- ต้องการร่างคำฟ้อง คำให้การ บัญชีพยาน
- ต้องการสร้างคดีใหม่ในระบบ

## Do not use this skill when

- ต้องการตรวจสอบเอกสารที่มีอยู่แล้ว → ใช้ `lawform-review`
- ถามเรื่อง Odoo ทั่วไปที่ไม่เกี่ยวกับคดี

## Instructions

1. อ่าน **agent-LAWYER.md** ในโปรเจกต์ก่อนเริ่มทำงานทุกครั้ง
2. ใช้ MCP tools (`odoo_create`, `odoo_search_read`, `odoo_write`, `odoo_execute`) เชื่อมต่อ Odoo
3. ทำตามขั้นตอนใน agent-LAWYER.md: รับเรื่อง → วิเคราะห์ → สร้างคู่ความ → สร้างคดี → สร้างเอกสาร → ร่างเนื้อหา
4. ห้ามปล่อยช่อง `...` ว่าง ต้องร่างเนื้อหาให้ครบทุกฟอร์ม
5. หลังสร้างเอกสารครบ ให้เรียก `action_apply_merge_fields` เพื่อ merge placeholder

## ชุดเอกสารมาตรฐาน

**คดีแพ่ง**: แบบ ๙, แบบ ๑, แบบ ๔, แบบ ๕, แบบ ๑๕
**คดีอาญา**: แบบ ๙, แบบ ๔, แบบ ๖, แบบ ๑๕
**ประกันตัว**: แบบ ๕๗, แบบ ๕๘ (ถ้ามี)
**อุทธรณ์**: แบบ ๓๒, แบบ ๓๓ (ถ้ามี)
