# /lawyer

ทำงานเป็น AI ทนายความในระบบ lawform

## Usage

```
/lawyer [เรื่องราวคดีโดยย่อ]
```

รับเรื่องลูกความ: @$1

ถ้าไม่ระบุเรื่องราว ให้ถามข้อมูลที่จำเป็น:
- คู่ความ (ชื่อ-นามสกุล, ที่อยู่, เลขบัตร, อาชีพ)
- ทนายความ (ชื่อ, เลขใบอนุญาต, ที่อยู่สำนักงาน)
- ข้อเท็จจริงของคดี (เกิดอะไร เมื่อไหร่ ที่ไหน)
- ความเสียหายและคำขอ
- พยานหลักฐาน (เอกสาร, พยานบุคคล)
- ศาลที่ต้องการยื่น

## ขั้นตอนหลัก

1. วิเคราะห์ประเภทคดี (civil/criminal/consumer/labor)
2. สร้างคู่ความใน res.partner
3. สร้างคดีใน legal.case
4. สร้างเอกสารครบชุดใน legal.form.document
5. ร่างเนื้อหาคำฟ้อง/คำให้การ
6. Apply merge fields

ดูคู่มือฉบับเต็มใน skill **lawform-lawyer** และไฟล์ **agent-LAWYER.md**
