# /review

ตรวจสำนวนคดีในระบบ lawform

## Usage

```
/review <case_id หรือ ชื่อคดี>
```

ตรวจสำนวนคดี: @$1

ถ้าไม่ระบุคดี ให้ค้นหาคดีล่าสุดจาก legal.case หรือถามว่าต้องการตรวจคดีไหน

## ตรวจ 6 หัวข้อ

| # | หัวข้อ | เกณฑ์ผ่าน |
|---|--------|---------|
| 1 | ชุดเอกสาร | ครบตาม case_type |
| 2 | Placeholder | ทุก placeholder แสดง ✓ |
| 3 | เนื้อหาอิสระ | ไม่มี `...` ที่ยังไม่ร่าง |
| 4 | บัญชีพยาน | มีทั้งพยานเอกสารและพยานบุคคล |
| 5 | คู่ความ | ชื่อ ที่อยู่ เลขบัตร ครบ |
| 6 | เลขไทย | U+0E50-0E59, พ.ศ. ถูกต้อง |

รายงานผลเป็นตาราง พร้อมรายการที่ต้องแก้ไขเรียงจากสำคัญมากไปน้อย

ดูคู่มือฉบับเต็มใน skill **lawform-review** และไฟล์ **agent-REVIEW.md**
